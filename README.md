## 🎯 Problem Statement

In today's world, people are more **connected digitally but more isolated socially**. Social media algorithms create echo chambers instead of fostering genuine human connections. ConnectImpact solves this by leveraging **AI** to match people with real-world communities, events, and volunteering opportunities that align with their interests and values.

---

## ✨ Key Features

### 🔍 **Community Discovery**
- AI-powered smart matching to find hobby groups, walking clubs, and local meetups
- Personalized recommendations based on interests, location, and availability
- Browse categories from fitness, reading clubs to startup networking

### 🤝 **Volunteer Matching**
- Connect with NGOs and causes that align with your skills and values
- Track volunteer hours and real-world impact
- Build a portfolio of social impact contributions

### 📅 **Event Booking**
- Discover local events (photography walks, networking nights, etc.)
- Real-time RSVP and availability tracking
- Smart event generation from natural language descriptions

### 💡 **Skill Exchange**
- Peer-to-peer knowledge sharing platform
- Trade skills: Teach guitar, learn coding, exchange languages
- Build new skills through community learning

### 🤖 **AI Wellness Companion (ConnectAI)**
- Personalized AI buddy that checks in on your mood
- Suggests local activities to reduce isolation
- Natural conversation starters to ease social anxiety
- Location-aware recommendations

### 📊 **Impact Dashboard**
- Visual analytics of volunteer hours logged
- Communities joined and events attended
- Social impact score tracking
- Gamified badges and achievement system (Explorer → Champion)

## 🚀 Quick Start

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- No backend setup required (static site)

### Local Development

1. **Clone the repository**
   ```bash
   git clone https://github.com/srivastavaarchit/connectimpact.git
   cd connectimpact
   ```

2. **Start a local server** (choose one)
   ```bash
   # Using Python 3
   python -m http.server 8000
   
   # Using Node.js (http-server)
   npx http-server
   
   # Using VS Code Live Server extension
   # Right-click index.html → "Open with Live Server"
   ```

3. **Open in browser**
   ```
   http://localhost:8000
   ```

4. **Explore the app**
   - Visit `/pages/communities.html` to see community discovery
   - Check `/pages/ai-assistant.html` for the AI companion demo
   - View `/pages/dashboard.html` for impact tracking
